## Features

<CardGrid stagger>
  <Card title="Blazing Fast" icon="rocket">
    Parallel segment downloading splits files into parts for maximum speed.
  </Card>
  <Card title="Daemon Architecture" icon="server">
    Background daemon manages downloads, enabling resume and multi-client support.
  </Card>
  <Card title="Cross-Platform" icon="laptop">
    Native support for Linux, macOS, and Windows with platform-specific optimizations.
  </Card>
  <Card title="Extensible" icon="puzzle">
    JavaScript extension system powered by Goja runtime for custom workflows.
  </Card>
</CardGrid>

## Quick Start

```bash
# Install (macOS)
brew install warpdl/tap/warpdl

# Download a file
warpdl https://example.com/file.zip
```
